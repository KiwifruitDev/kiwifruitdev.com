# Language

In Leaflet, there is a written language that masks real-world English. This language is called **Terran**.

## Lore

Terran is canonically a rune-based language that is written vertically.

For speaking, Terran is heavy on clicks and pops unlike most real-world languages.

Traditionally, Terran is written using stamps made of wood or stone.

## Do nots

- Terran is read as English for story purposes. It is not meant to be read as English.
- Ink is not allowed to be used to write Terran. Ink harvesting doesn't exist in Lightly.

## Alphabet

The language is based on the [Futhark](https://en.wikipedia.org/wiki/Futhark) rune set, but with notable exceptions that look too close to English letters.

If a rune looks too close to an English letter, it is replaced with a random character.

This is to make it as difficult as possible to transliterate the language into English, and to prevent players from accidentally reading the language as English.
